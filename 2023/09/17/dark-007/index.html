如何快速搭建制作一个暗网网站？

发表于2023-09-17|更新于2023-09-17|暗网
|阅读量:

快速搭建一个暗网网站和暗网网站搭建和暗网域名申请方法

众所周知，暗网网站是一个安全和隐蔽的网站，可以保护你的个人隐私，防止IP等信息被追踪到，可以安全的进行你所喜欢的而且隐蔽保密的事情。

搭建一个暗网网站有以下几个步骤：

申请域名

apt 一键安装

可以通过 apt 直接安装 tor 服务

apt install tor

如果不行请进行以下操作

手动安装

修改 source.list

sudo vi /etc/apt/sources.list
最后加下面这两行

添加密钥

安装

dns 选择

这里默认是 deb.torproject.org
备选的有 deb-master.torproject.org、mirror.netcologne.de、tor.mirror.youam.de

编辑 tor 配置文件

vi /etc/tor/torrc
添加

启动 tor

sudo service tor start
一启动，那么你的域名也就生成了

查看域名

cat /var/lib/tor/hidden_service/hostname

域名的结尾一定是onion，代表着洋葱网络
真正的安全网络，很难被追踪

如何挑选一个心仪的onion域名

.onion 的域名生成是私钥 >（rsa）> 公钥 >（sha1）> 杂凑值 >（base32 编码）> 得到很丑的地址.onion。这道理跟比特币生成特定地址是同样的，但是因为计算量太大所以大家都会折衷，只要地址前几位有个人特色就好。

所以原理是，使用高性能运算机器，不停的一值尝试不同的私钥输入，看看有没有满意的结果，例如 Facebook 的洋葱地址 “facebookcorewwwi.onion”。相当的漂亮，相当的好看，要爆破这么多位，大概也只有 Facebook 等拥有超大规模服务器集群的厂商可以做得到，因为在密码学上的评估要暴力解出这样的结果以 1.5Ghz 处理器要 260 万年，惊为天人 Facebook 展现出庞大运算的惊人实力。这是暗网目前最漂亮的域名了。

项目地址：

https://github.com/katmagic/Shallot

https://github.com/lachesis/scallion

https://github.com/cathugger/mkp224o

如果运行过程中缺了什么套件沒有办法运行

可以输入以下命令：

sudo apt-get install libssl-dev

网站代码编写

网站代码的编写不限制语言、框架和技术，甚至你可以制作静态网站

最为重要的一点是，不要使用js尽量，使用js可能让泄露信息，因此不要使用js

端口是你通过命令配置的端口
